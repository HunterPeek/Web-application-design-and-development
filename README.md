# Лабораторна робота 2 — HTML форма з CSS та JS

## Мета
- Створити форму у HTML.
- Стилізувати форму за допомогою CSS.
- Додати перевірку даних через JavaScript.

## Форма
Поля:
- Ім’я
- Email
- Пароль
- Повтор пароля

Кнопка **Надіслати**.

## Перевірка даних
- Ім’я не порожнє
- Email у правильному форматі
- Пароль ≥ 6 символів
- Повтор пароля збігається

## Файли
- `form.html` — HTML форма
- `style.css` — стилі
- `script.js` — перевірка даних

## Запуск
Відкрити `form.html` у браузері.
